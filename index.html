About Me

The year was 2015 when I first got exposed to web programming through a popup on some website. I swiftly learned HTML and wrote my resume in it. That was cool and I wanted to learn more but it seemed so difficult. I dropped it.

I stumbled upon coding again in 2019, and that was when I challenged myself a bit harder. I signed up for the freeCodeCamp Response Web Design course and properly learned how to structure web pages.

I built a portfolio and began making WordPress websites for organizations in Sierra Leone and Australia.

Fast-forward to February this year, I gained a renewed interest in delving even deeper into the web by studying JavaScript, React, Sass, Bootstrap, Node and Express.

I am well on my way to fullstack using the MERN stack.

Feel free to connect with me.
